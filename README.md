# My Playlist - Spotify Clone

A simple and interactive **Spotify-inspired music player** built using **HTML, CSS, and JavaScript**. This project allows users to play songs, switch between tracks, view cover images, and control playback with a clean web-based interface.

## Features

- Play and pause songs
- Next and previous track controls
- Interactive progress bar
- Dynamic song list
- Cover images for each track
- Animated now playing indicator
- Simple Spotify-style user interface

## Tech Stack

- HTML5
- CSS3
- JavaScript

```

## How It Works

- The music player loads songs using JavaScript
- Each song item displays a song name and cover image
- Users can play a specific song from the playlist
- The progress bar updates while the song is playing
- Users can move to the next or previous track using controls

## Features in Detail

- Real-time song playback
- Playlist-based song selection
- Auto-updating song title display
- Custom cover art for every song
- Smooth and beginner-friendly UI

## Getting Started

1. Clone or download this repository
2. Open the project folder
3. Make sure all songs and cover images are inside the correct folders
4. Open `index.html` in your browser
